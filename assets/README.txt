Drop your images here with these EXACT filenames.
The website will automatically use them instead of the
hand-drawn placeholder art — no code changes needed.

kingdom-night.jpg      -> Kingdom of Endless Night (Norway / aurora)
kingdom-stories.jpg    -> Hall of Stories (Paris / the Louvre)
kingdom-tomorrow.jpg   -> City Beyond Tomorrow (Dubai / Future Museum)
kingdom-rain.jpg       -> Garden of Rain (Singapore)
kingdom-lake.jpg       -> Lake Above the Clouds (Chandrataal)

Notes:
- .jpg, .png, or .webp all work — just rename the file
  to match exactly (e.g. kingdom-night.jpg), or edit the
  "image" path for that kingdom in js/data.js if you'd
  rather keep your own filename/extension.
- Any image size or shape works. Landscape (wider than
  tall) will look best, especially for kingdom-night,
  kingdom-tomorrow, and kingdom-lake, which go full-width.
- Until a file is added, each kingdom shows its hand-coded
  placeholder illustration automatically.
